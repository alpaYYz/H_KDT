프로젝트3-딥러닝

설명

먼저, 2017년부터 2021년까지 카테고리별 금융범죄 현황 데이터를 경찰청에서 가져와 분석해본 결과 메신저 피싱 건수가 2019년부터 급증하기 시작해서 2021년 최고점을 찍은 것을 확인할 수 있었습니다.

설명

메신저 피싱이 가장 많았던 2019년부터 2021년까지의 금융범죄 현황을 파이차트로도 시각화해본 결과 메신저 피싱이 50%를 훨씬 넘게 차지한다는 것을 볼 수 있었습니다.

설명

저희가 AI Hub에서 수집한 일상 대화 데이터에서의 단어 빈도수 Top 20을 뽑아서 시각화해보았습니다. 그 결과, 게임, 요즘, 생각, 나라, 코로나 등의 단어가 가장 많이 나온 것을 확인할 수 있었습니다.

설명

피싱 메시지 데이터는 찾아봐도 마땅한 데이터가 없어서 102개의 피싱 메시지 데이터를 직접 만들어서 단어 빈도수 Top 20을 뽑아서 시각화해보았습니다. 그 결과, 엄마, 부탁, 문자, 수리, 번호, 이체 등의 단어가 많이 나온 것을 확인할 수 있었습니다.

설명

LSTM 모델의 정확도를 돌려본 결과 정확도 94.37%의 결과가 나왔습니다.

설명

LSTM 모델을 에폭 300으로 하여 돌려본 결과를 그래프로 시각화해보았습니다.

설명

LSTM 모델의 결과로 단순히 '엄마' 라는 단어와 피싱 메시지로 의심되는 문장을 넣어본 결과, 0-피싱 메시지가 아님, 1-피싱 메시지 로 분류되는 결과입니다.

설명

RNN 모델로 돌려본 결과, 정확도 53.45%의 결과가 나왔습니다.

설명

RNN 모델을 에폭 500으로 하여서 돌려본 결과를 그래프로 시각화해보았습니다.

설명

Logistic Regressing인 머신러닝 모델로도도 돌려본 결과 Train 정확도는 약 96%로 나왔고, Test 정확도는 약 95%로 나왔습니다.

설명

Logistic Regressing 머신러닝 모델로 필터링을 한 결과 피싱 메시지로 의심되는 '엄마 폰액정이 깨져서 송금해줘' 문장을 1(피싱메시지)로 잘 분류가 되었습니다.

설명

마찬가지로 피싱 메시지가 아닌 '오늘 영화 진짜 재밌었다' 라는 문장을 0(피싱메시지가 아닌 일상 대화 메시지)로 잘 분류가 되었습니다.

설명

Naive Bayes라는 머신러닝 모델로도 돌려본 결과 Train 정확도는 약 96%이고, Test 정확도는 약 89%로 나왔습니다.

설명

Naive Bayes 머신러닝 모델로 필터링을 한 결과 피싱 메시지로 의심되는 '엄마 바빠요? 내가 급해서 그런데 카카오톡 친구추가해서 이 계좌로 송금 좀 해줘' 문장을 1(피싱메시지)로 잘 분류가 되었습니다.

설명

원래는 피싱메시지 데이터를 저희가 직접 만들어서 엄마라는 단어를 많이 넣어서 엄마라는 단어만으로도 피싱메시지로 분류가 되었는데, 해결하고나니 0(피싱메시지가 아닌 일상대화 메시지)로 잘 분류가 되었습니다.

설명

각 모델들의 Train 정확도와 Test 정확도입니다.
